## Deep Thought

Prompts the user for the answer to the Great Question of Life, 
the Universe and Everything. Outputs `Yes` if the user inputs 
`42`, `forty-two`, or `forty two` (case- and space-insensitive). 
Otherwise outputs `No`.

### Concepts practiced
- User input handling
- String normalization with `.lower()` and `.strip()`
- Conditional logic with `in`
