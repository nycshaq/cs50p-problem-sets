## Bank

Prompts user for a greeting. If the greeting starts with "Hello", 
outputs `$0`. If it starts with "h" but isn't "hello" (e.g. "hey", 
"howdy"), outputs `$20`. Any other greeting outputs `$100`.

### Concepts practiced
- User input handling
- String normalization with `.lower()` and `.strip()`
- Conditional logic with `.startswith()`
